# Модифицированный иерархический алгоритм построения триангуляции Делоне 📐

Интерактивная реализация двухпроходного алгоритма (модифицированный иерархический алгоритм) для построения 
**триангуляции Делоне** на плоскости. Программа позволяет визуально наблюдать каждый этап: вставку точек без проверки 
условия Делоне (проход 1) и последующее исправление триангуляции с помощью флипов (проход 2).

## Цель проекта 🎯

- Продемонстрировать работу **двухпроходного алгоритма** построения триангуляции Делоне, описанного в книге 
  [**А.В. Скворцова «Алгоритмы построения и анализа триангуляции»**](util/SkvortsovAV.pdf) (глава 4, §4.2).
- Предоставить наглядный интерактивный инструмент для изучения геометрических алгоритмов.
- Реализовать базовые геометрические предикаты (`orient2d`, `in_circle`, `pt_in_tri`) и визуализацию на `matplotlib`.

## Описание 📎

Алгоритм работает в два этапа:

1. **Первый проход (быстрая вставка)**
    - Строится большой **супертреугольник**, охватывающий все исходные точки.
    - Каждая точка последовательно вставляется в текущий треугольник, который её содержит.
    - Треугольник удаляется, вместо него создаются три новых, соединяющих точку с тремя вершинами старого треугольника.
    - Вставка выполняется **без проверки условия Делоне** — целью является лишь получение некоторой триангуляции.

2. **Второй проход (глобальное перестроение)**
    - Исходная триангуляция (включая супервершины) подвергается многократному просмотру всех внутренних рёбер.
    - Для каждого ребра `(a,b)`, общего для двух треугольников `(a,b,c)` и `(a,b,d)`, проверяется условие Делоне: 
      не лежит ли точка `d` внутри окружности, описанной вокруг `(a,b,c)`.
    - Если условие нарушено, выполняется **флип** — ребро `(a,b)` заменяется на `(c,d)`, а треугольники перестраиваются.
    - Процесс повторяется, пока не останется ни одного нарушения.
    - В конце удаляются все треугольники, содержащие хотя бы одну вершину супертреугольника.

**Возможности интерактивной 🤳 программы:** 

- Добавление / удаление точек мышью или по координатам.
- Навигация по шагам:
    - Кнопки **«Проход 1»** и **«Проход 2»** для переключения между этапами.
    - Клавиши **←** / **→** (или **A** / **D**) для просмотра каждого флипа на втором проходе.
- Отображение описанных окружностей (клавиша **H**) для проверки условия Делоне.
- Подсветка треугольника под курсором мыши.
- Генерация случайного набора точек (клавиша **R**) и очистка (**C**).
- Визуализация суперструктуры (большого вспомогательного треугольника) в первом проходе.
