# Многопоточное программирование

В данном подпроекте я реализовывал распараллеливание алгоритмов для решения математических задач. Проект был создан в
ходе
лабораторных работ, которые были предложены по курсу "Современные технологии многопоточных вычислений".
В проекте находится учебное пособие и практикум авторства Погребняка М. А, который является преподавателем по
дисциплине.
Варианты для лабораторных работ также выдавал преподаватель. Большая часть информации описана в учебном пособии.

## Структура

* **Elementary**: предварительные программы чтобы "вкатиться в работу";
* **Matrix**: параллельное умножение матриц;
* **Integral**: параллельное численное интегрирование;
* **Equation**: параллельное численное решение для уравнения теплопроводности.

Также есть два .pdf файла собственно само учебное пособие, а также практикум.

## Предварительно

Для компиляции кода, написанного на C++, я использую IDE **Visual Studio 2022**.

Для компиляции и интерпретации кода, написанного на Python, я использую IDE **PyCharm 2025.1.1.1**.

### Инструмент MPI

С официального сайта Microsoft **https://www.microsoft.com** необходимо скачать **msmpisdk.msi** и **msmpisetup.exe**.
После этого должны установиться переменные окружения: **MSMPI_INC**, **MSMPI_BIN**, **MSMPI_LIB64** и **MSMPI_LIB32**.
В консоли (на Windows) если вызвать команду ниже, то должна вывестись на экран версия и возможные параметры.
Я использую версию **10.0.12498.5** в своих примерах.

```shell

mpiexec

```

### Подключение MPI к VS

1) Для того чтобы подключить MPI в VS следует открыть проект в IDE. Затем сверху на панели нажать **Проект** -> *
   *Свойства**
   -> **Каталоги VC++**, найти секцию **Внешние каталоги включения** и добавить туда в конец путь **\$(MSMPI_INC);**.

2) Далее переходим там же в **Компоновщик** -> **Общие** и в секции **Дополнительные каталоги библиотек** добавляем в
   конец
   путь **\$(MSMPI_LIB64);**.

3) Теперь переходим **Компоновщик** -> **Ввод** и в секции **Дополнительные зависимости** добавляем в конец
   сами библиотеки **msmpi.lib;msmpifmc.lib;msmpifec.lib;**.

4) Осталось нажать **Применить** и **Ок**.

### Запуск MPI

Запустить программу с помощью инструмента MPI можно так:

```shell

# Или любое другое количество процессов
mpiexec -n 16 MyProject.exe 

```

### Запуск MPI из VS

Для облегчения работы можно запускать файл командой MPI из VS. Для этого нужно перейти в верхней панели проекта *
*Средства**
-> **Внешние инструменты** и далее добавить новый инструмент:

* Название: Run with MPI;
* Команда: C:\Program Files\Microsoft MPI\Bin\mpiexec.exe;
* Аргументы: -n 16 $(TargetName).exe (или сколько вы хотите потоков);
* Исходный каталог: $(BinDir);
* Поставьте галочку для "Запросить аргументы" (частенько приходиться менять количество процессов).
* Осталось только нажать "Ок".

Теперь можно запускать так: **Средства** -> **Run with MPI**.

### Включение OpenMP

Для того чтобы включить в проекте OpenMP необходимо перейти опять в свойства проекта, а затем перейти **C/C++** -> *
*Язык**,
найти секцию **Поддержка Open MP** и установить флажок **да (/openmp)**. Это всё.

### Matplotlib

Для отрисовки графиков используется библиотека **matplotlib**, которую можно скачать прямо в GUI IDE.

### Дополнительно

Напомню также, что, для того чтобы запускать несколько файлов .cpp с методом main можно пользоваться средством
исключения
