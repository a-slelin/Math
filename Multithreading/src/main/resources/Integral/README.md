# Численное интегрирование

## Дано

Даны четыре определённых интеграла от a) до d) соответственно:

1) Интеграл от 0 до 1 для функции f(x) = exp(x) * sin(x);
2) Интеграл от pi/4 до 3pi/4 для функции f(x) = cos(sqrt(x)) / sqrt(x);
3) Интеграл от 0.1 до 0.3 для функции f(x) = x * ln((1 + x) / (1 - x));
4) Интеграл от pi до 3pi для функции f(x) = (x * cos(x)) / (pow(sin(x), 3) + 2).

## Найти

Каждый интеграл требуется численно вычислить 3 методами: прямоугольника, трапеции и симпсона с точностью 10^-10.
Результаты записать в файл.

## Структура

* **Program**: программа на C++.
* **Graphic**: программа на Python для сбора и отрисовки данных.
* **Results**: результаты лабораторной работы.
