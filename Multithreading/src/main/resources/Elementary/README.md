# Элементарные программы

Здесь написаны простенькие программы с использованием MPI и OpenMP, для того чтобы разобраться с этими инструментами.

## Структура

1) Hello, world! - программа.
* **single_hello_world.cpp** - однопоточная программа;
* **mpi_hello_world.cpp** - MPI программа;
* **openmp_hello_world.cpp** - OpenMP программа.

2) Деление процессов по подгруппам в MPI.
* **mpi_subgroups.cpp**.

3) Кольцевая пересылка в MPI.
* **mpi_circle.cpp**.

4) Параллельный подсчёт суммы с MPI.
* **mpi_sum.cpp**.

5) Версия OpenMP.
* **openmp_version.cpp**.

6) Задание количества потоков в OpenMP.
* **openmp_num_threads.cpp**.

7) Информация о потоках в OpenMP.
* **openmp_info_threads.cpp**.

8) Работа с данными в OpenMP.
* **openmp_value.cpp**.

9) Работа с секциями в OpenMP.
* **openmp_section.cpp**.

10) Директива critical в OpenMP.
* **openmp_critical.cpp**.

11) Reduction в OpenMP.
* **openmp_reduction.cpp**.

12) Работа с schedule в OpenMP.
* **openmp_schedule.cpp**.
