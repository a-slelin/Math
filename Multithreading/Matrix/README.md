# Умножение матриц

## Дано

Есть две матрицы **A** и **B** размерами 1000x1000.

Элемент матрицы **A** со строкой i и столбцом j рассчитывается так: a(i, j) = (2 * i + 1)^2 + (2 * j + 1)^2.

Элемент матрицы **B** со строкой i и столбцом j рассчитывается так: b(i, j) = -i * j.

## Найти

Необходимо вычислить матрицу **C** = **A** * **B**. Результат записать в файл.

## Структура 

* **Program**: программа на C++.
* **Graphic**: программа на Python для сбора и отрисовки данных.
* **Results**: результаты лабораторной работы.
