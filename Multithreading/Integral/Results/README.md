# Результаты

## Содержание

* **single_{x}.txt**: результаты выполнения однопоточного алгоритма для интеграла x.
* **openmp_{x}.txt**: результаты выполнения многопоточного алгоритма с OpenMP для интеграла x.
* **mpi_{x}.txt**: результаты выполнения многопоточного алгоритма с MPI для интеграла x.
* **openmp_{x}.png**: график для OpenMP для интеграла x.
* **mpi_{x}.png**: график для MPI для интеграла x.
* **integral_{x}.png**: подсчёт интеграла x на калькуляторе.


## Дополнительно

* Из-за того что метод прямоугольника слишком неточен очень тяжело добиться точности 10^-10, поэтому в примерах
я использую точность 10^-8 для интегралов a), b), c) и 10^-6 для интеграла d). Конечно это не влияет на общее понимание 
работы и решения задачи, просто мой ноут сказал мне пока при 10^-10 (Оператива 16 гигов + Райзен 7 на 16 потоков).
