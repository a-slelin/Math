# Программа на C++

## Содержание

* **general.h**: заголовочный файл основных констант и функций.
* **general.cpp**: файл с основными константами и функциями.
* **single.cpp**: реализация однопоточного алгоритма.
* **openmp.cpp**: реализация многопоточного алгоритма с OpenMP.
* **mpi.cpp**: реализация многопоточного алгоритма с MPI.
