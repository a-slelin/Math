# Результаты

## Содержание

* **single.txt**: время выполнения однопоточного алгоритма.
* **openmp.txt**: время выполнения многопоточного алгоритма с OpenMP.
* **mpi.txt**: время выполнения многопоточного алгоритма с MPI.
* **single_matrix.txt**: результаты выполнения однопоточного алгоритма.
* **openmp_matrix.txt**: результаты выполнения многопоточного алгоритма с OpenMP.
* **mpi_matrix.txt**: результаты выполнения многопоточного алгоритма с MPI.
* **openmp.png**: график для OpenMP.
* **mpi.png**: график для MPI.
* **graph.png**: сравнение графиков MPI и OpenMP.
