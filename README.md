# Математика 🧮

Цель данного репозитория состоит в решении некоторых математических задач.

## Структура проекта 📁

Репозиторий организован по следующим разделам:

* **Absorption** 📸: вычисление и визуализация коэффициентов поглощения **n**-мерным симплексом **n**-мерного куба.

* **Chaos** 💥: пример с реализацией хаотической динамики на примере логистического уравнения с параметром **r = 4**.

* **Multithreading** 🔀: многопоточное программирование на **C++** с использованием **MPI** и **OpenMP** для решения
  некоторых математических задач, таких как умножение матриц, численное интегрирование и численное решение уравнения
  теплопроводности.

* **Triangulation** 📐: интерактивное приложение, реализующее триангуляцию Делоне с использованием модифицированного
  иерархического алгоритма.

### Инициализация проекта

Вначале необходимо, чтобы maven подгрузил все необходимые зависимости; к тому же, рекомендую установить зависимости в
свой локальный репозиторий .m2:

```shell
# Клонируем репозиторий;
git clone https://github.com/a-slelin/Math.git;

# Переходим в корневую папку;
cd Math;

# Собираем проект с помощью maven;
mvn clean install;
```

После чего необходимо настроить окружение для python-интерпретатора:

```shell
# Создаём виртуальное окружение;
python -m venv .venv;

# Переключаемся на виртуальное окружение;
.\.venv\Scripts\activate.bat;

# Устанавливаем необходимые библиотеки;
pip install -r requirements.txt;
```
