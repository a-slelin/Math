# Математика

Этот репозиторий содержит решение разных математических задач на Java (и не только).  
Цель проекта — демонстрация решения математических задач.

## Структура проекта

Репозиторий организован по следующим разделам:

* **Chaos**: пример с реализацией хаотической динамики на примере логистического уравнения с параметром 4.

* **Multithreading**: многопоточное программирование на C++ с использованием MPI и OpenMP для решения некоторых математических задач,
таких как умножение матриц, численное интегрирование и численное решение уравнения теплопроводности.
